How to run locally --

> Step 1. Download / Clone this repo

> Step 2. Run npm i / yarn

> Step 3. Run npm start / yarn start

---

For checking test --

> Run npm test / yarn test

---

Unit test and integration test has been implemented for Redux action, reducer and Redux store.
